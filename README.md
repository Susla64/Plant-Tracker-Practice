# Облік кімнатних рослин (Plant Watering Tracker) 🌿

Консольна утиліта мовою C++ для автоматизації відстеження графіка поливу кімнатних рослин. Проєкт розроблено в рамках проходження навчальної практики.

## 📌 Функціональні можливості
* **Додавання рослин:** Запис нової рослини із зазначенням назви, дати останнього поливу та необхідної частоти догляду (у днях).
* **Перегляд списку:** Виведення всіх наявних у базі рослин із детальною інформацією.
* **Розумний розрахунок:** Автоматичне визначення рослин, які потребують поливу **саме сьогодні** (на основі системної дати ПК).
* **Видалення:** Зручне видалення рослин зі списку за їхнім порядковим номером.
* **Збереження даних:** Автоматичне збереження та завантаження списку рослин із локального текстового файлу (`plants_data.txt`).

## 🛠 Технології
* Мова програмування: **C++**
* Стандартні бібліотеки: `<iostream>`, `<fstream>`, `<vector>`, `<string>`, `<ctime>`, `<sstream>`.

## 🚀 Як запустити проєкт
1. Завантажте файли `main.cpp` та `plants_data.txt` (якщо є база даних) у спільну папку.
2. Скомпілюйте файл `main.cpp` за допомогою будь-якого компілятора C++ (наприклад, GCC, Clang або MSVC).
3. Запустіть отриманий виконуваний файл. 
4. *Примітка: Програма створює файл `plants_data.txt` автоматично при першому успішному збереженні даних, якщо він був відсутній.*
